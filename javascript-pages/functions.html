Functions

Functions are pieces of code that can be reused by calling them multiple times (very often with different parameters to archieve different results).

When creating a function, two keywords come in handy. First one, function defines the function. Second one is optional, but present in most functions - the return keyword.

The return keyword immediately ends the execution of the function, and passes the result to the code that called it.

Example of a function without parameters, that always returns 1 to the code that called it.

function exampleFunc() {
  return 1
}

Function can have no parameters, but most functions will have multiple parameters.

function sumTwo (a,b) {
  return a + b
}

The code above demands two parameters to be passed and returns the sum of them

Here is how that function might be used

var x;
x = sumTwo(5,10)
x == 15 //"true"

If a function creates any variables, they aren't accesible outside that function.

function testFunc() {
  var x;
  x = 10;
  // right here, the variable x is accesible, because it's in the same code block
}
// but here, the variable x is not accesible, because it "belongs" to the function "testFunc" code block

If a function is called without parenthesis, no code defined within function will be executed, because calling a function like that only refers to the variable of type function with the name of said function

function myFunc() {
  return 5
}
typeof myFunc() // returns "number", because code in function block got executed and returned 5
typeof myFunc // returns "function", because myFunc only refers to a variable called like that

Variables and operators

Variables store data values and operators are used to compute or assign values to variables

Variables

Variables are declared using the var keyword

var x;

All variables must be declared before using them later in the code, and every variable must have an unique name (case sensitive).

Multiple variables can be declared in one line

var x,y,z;

Variable that has been declared but hasn't been assigned any value yet, will have the undefined value

var x; //x holds undefined value until any other value is assigned to it

Operators

Operators are used to assign values to variables and perform arithmetic on them

The most basic operator is the assignment operator, which assigns values to variables

var x,y;
x = 5;
y = 10;

Variables can be declared and have a value assigned to them in a single line

var x=5, y=10;

Arithmetic operators allow us to perform mathematical operations on variables

var x;
x=10;
x = x + 5;  // x=15
x = x - 7;  // x=8
x = x / 4;  // x=2

There are also arithmetic assignment operators that both perform arithmetic and assign a value to the variable.

This example

var x;
x += 5;

Has the same result as this one

var x;
x = x + 5;

List of arithmetic assignment operators

x += y
is the same as x = x + y
x -= y
is the same as x = x - y
x *= y
is the same as x = x * y
x /= y
is the same as x = x / y
x **= y
is the same as x = x ** y
x %= y
is the same as x = x % y

Data types

Variables can hold one of many different types of data listed below

  • Primitive types
    • number
    3, 5.6 or -19
    string
    "a word", "50"
    boolean
    true or false
    undefined
    returned when a variable has no value assigned
  • Complex types
    • object
    object, array or null type
    function
    functions to be invoked later in the code

Primitive data values are the simplest types of data which don't have additional properties.

Objects and functions are complex data values which have additional methods and properties.

Variable types can be identified by using the typeof operator

var x,y,z;
x = 10
y = "pizza"
z = true

typeof x //"number"
typeof y //"string"
typeof z //"boolean"

Script placing

Most JavaScript scripts are placed into a HTML document by using one of three different ways

In event attribute

Some of HTML elements have incredibly handy attributes called event attributes, which values are executed when a certain event takes place. For example, onclick attribute of a button is activated when the button is being clicked by the user. This way, we can execute JavaScript code as a response to an event. 

<button type="button" onclick="document.getElementById('demo').innerHTML = 'Hello World!'">
  Click me!
</button>

This script changes a paragraph when the button is clicked

In script element

Scripts can also be placed within <script> element, in order to avoid writing long scripts into event attributes. 

<button type="button" onclick="myFunction()">
  Click me!
</button>

<script>
function myFunction() {
  document.getElementById('demo').innerHTML = "Hello world!"
}
</script>

Script elements can be placed anywhere in the HTML document, either in body or head section but it's recommended to put them at the end of the body section, because loading scripts slows down the display of the page

In external file

This method is the best one, because we can separate JavaScript and HTML, which makes the page easier to maintain. External scripts can be imported by using src attribute of the script element. 

<script src="script.js"></script>

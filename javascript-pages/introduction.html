Javascript

Javascript makes pages more dynamic and interactive

We can define a client-side script (JavaScript) by using <script> tag

Sample script, which writes some text into any HTML element with id="text-container"

<script>
  document.getElementById("text-container").innerHTML = "Hello World!";
</script>

JavaScript can also change attributes and styles of chosen HTML element

document.getElementById("sample-element").style.color = "blue";
document.getElementById("sample-image").src = "picture.jpg";

JavaScript function can be activated by some events, for instance onclick event on a button

<button type="button" onclick="document.getElementById('date-button').innerHTML = Date()">Date button</button>
<p id="date-button">[output of the script]</p>

Noscript

<noscript> tag allows us to define alternative content, when JavaScript couldn't be loaded,
similar to alt attribute in <img> tag

<script>
  document.getElementById("text-container").innerHTML = "Hello World!";
</script>
<noscript>Your browser does not support JavaScript!</noscript>

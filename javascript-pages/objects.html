Objects

Objects are variables that can contain many values.

Object can have properties - simple values, and methods - functions that can be performed

Example of a simple object

var pizza = {ingredient:"pepperoni", sauce:"tomato"}

Object above has two properties (ingredient and sauce), but no methods

Properties can be accessed in two ways

objectName.propertyName

objectName["propertyName"]

Methods are a special case of a property. They are actions that can be performed on the object. Method is a function that is an object's property.

Example

var person = {
  name: "Phil",
  favouriteLanguage: "PHP",
  introduction: function() {
    return this.name + "'s favourite programming language is " + this.favouriteLanguage;
  }
};

[output of the script]

Methods are functions belonging to an object, so in order to get their returned value, they must be called with parenthesis.

Methods can manage and access object's properties by using the this keyword. It refers to the owner of the properties (the object). Methods use this keyword instead of objectName.propertyName syntax because they "belong" to the same object as properties they want to access. Therefore, using this keyword outside objects is incorrect.

Iframe

Iframe is used to display a page in a page


Basic iframe attributes

  • Height
  • Width
  • Title
  • Name - used to target the iframe by a link
  • Style
    • Height
    • Width
    • Border

Targeting the iframe

Sample <a> link targeting the iframe

<a href="old_home.html" target="myIframe">Old home page</a>
<a href="home.html" target="myIframe">New home page</a>

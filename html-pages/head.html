Head

The <head> is a place for information about the webpage and it's not visible for the user in most cases

Title

The title is visible in the browser's tab and in search engines' results

    <head>
        <title>The title of the website</title>
    </head>

Style

The style element defines style information for the page

    <head>
        <style>
            body {background-color: blue;}
            h1 {color: red;}
        </style>
    </head>

Link

The link element is used to define relationships between the document and external files, mostly stylesheets

    <head>
        <link rel="stylesheet" href="style_file.css">
    </head>

Meta

The meta element specifies the character set, keywords or page description and other data
often used by search engines or web services


Defining the character set

    <head>
        <meta charset="UTF-8">
    </head>

Defining a description

    <meta name="description" content="My webpage">

Setting the viewport

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

The viewport is the user's visible area of a web page, and it changes with the device - smaller on a phone, bigger on a computer screen. width=device-width sets the width of the page to width of the device screen. initial-scale=1.0 sets the initial zoom level when the page is loaded

Script

Script element defines client-side JavaScript scripts

    <script>
        function myFunction() {
            document.getElementById("sample-element").innerHTML = "Hello World!";
        }       
    </script>

Base

Base element defines the base URL used by realative URLs. It must have an href or target attribute

    <head>
        <base href="https://mywebsite.com" target="_blank">
    </head>

Only one <base> element can be present in a single page
